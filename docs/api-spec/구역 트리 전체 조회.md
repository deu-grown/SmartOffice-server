# 구역 트리 전체 조회

카테고리: 구역
설명: 전체 구역을 계층형 트리 구조로 조회한다. 최상위 구역(parent_id=NULL)을 루트로 하위 구역을 중첩하여 반환한다.
Method: GET
URL: /api/v1/zones/tree
담당자: 박성종
사용자: ADMIN
우선순위: 4

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 구역 ID | Long |  | X | 1 |
| data[].name | 구역명 | String |  | X | "1층" |
| data[].zoneType | 구역 유형 | String |  | X | "FLOOR" |
| data[].description | 구역 설명 | String |  | O | "1층 전체 구역" |
| data[].parentId | 상위 구역 ID | Long |  | O | null |
| data[].children | 하위 구역 목록 | Array |  | X | [...] |
| data[].children[].id | 하위 구역 ID | Long |  | X | 2 |
| data[].children[].name | 하위 구역명 | String |  | X | "1층 회의실 A" |
| data[].children[].zoneType | 하위 구역 유형 | String |  | X | "AREA" |
| data[].children[].description | 하위 구역 설명 | String |  | O | "소회의실" |
| data[].children[].parentId | 상위 구역 ID | Long |  | X | 1 |
| data[].children[].children | 하위 구역 목록 | Array |  | X | [] |

### Status

| status | response content |
| --- | --- |
| 200 | 구역 트리 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
