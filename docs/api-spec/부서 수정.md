# 부서 수정

카테고리: 부서
설명: 부서 정보를 수정한다.
Method: PUT
URL: /api/v1/departments/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 3

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 부서명 | String |  | X | "개발팀" |
| description | 부서 설명 | String |  | O | "풀스택 개발 담당" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "name": "개발팀",
  "description": "풀스택 개발 담당"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "부서 정보가 수정되었습니다." |
| data.id | 부서 ID | Long |  | X | 1 |
| data.name | 부서명 | String |  | X | "개발팀" |
| data.description | 부서 설명 | String |  | O | "풀스택 개발 담당" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "부서 정보가 수정되었습니다.",
  "data": {
    "id": 1,
    "name": "개발팀",
    "description": "풀스택 개발 담당",
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 부서 수정 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 부서 |
| 409 | 이미 존재하는 부서명 |
