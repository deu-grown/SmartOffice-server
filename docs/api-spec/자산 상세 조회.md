# 자산 상세 조회

카테고리: 자산관리
설명: 특정 자산의 상세 정보를 조회한다.
Method: GET
URL: /api/v1/assets/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 8

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.id | 자산 ID | Long |  | X | 1 |
| data.assetNumber | 자산 번호 | String |  | X | "ASSET-001" |
| data.name | 자산명 | String |  | X | "맥북 프로 16인치" |
| data.category | 자산 분류 | String |  | X | "ELECTRONICS" |
| data.rfidUid | RFID 태그 UID | String |  | O | "RF:A1:B2:C3" |
| data.allowedZoneId | 허용 구역 ID | Long |  | O | 2 |
| data.allowedZoneName | 허용 구역명 | String |  | O | "2층 개발팀" |
| data.assignedUserId | 담당자 직원 ID | Long |  | O | 1 |
| data.assignedUserName | 담당자 이름 | String |  | O | "박성종" |
| data.description | 자산 설명 | String |  | O | "팀장 전용 맥북" |
| data.status | 자산 상태 | String |  | X | "ACTIVE" |
| data.purchasedAt | 구입일 | String |  | O | "2025-01-15" |
| data.createdAt | 등록일시 | String |  | X | "2026-03-02T09:00:00" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

### Status

| status | response content |
| --- | --- |
| 200 | 자산 상세 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 자산 |
