# 주차면 등록

카테고리: 주차
설명: 새로운 주차면을 등록한다. deviceId는 초음파 센서와 1:1 매핑이므로 이미 다른 주차면에 연결된 장치는 등록 불가.
Method: POST
URL: /api/v1/parking/spots
담당자: 우하
사용자: ADMIN
우선순위: 6

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| zoneId | 주차장 구역 ID | Long |  | X | 5 |
| spotNumber | 주차면 번호 | String |  | X | "C-01" |
| spotType | 주차면 유형 | String |  | X | "NORMAL" |
| deviceId | 초음파 센서 장치 ID | Long |  | O | 20 |
| positionX | 지도상 X 좌표 | Integer |  | O | 200 |
| positionY | 지도상 Y 좌표 | Integer |  | O | 300 |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "zoneId": 5,
  "spotNumber": "C-01",
  "spotType": "NORMAL",
  "deviceId": 20,
  "positionX": 200,
  "positionY": 300
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "주차면이 등록되었습니다." |
| data.id | 생성된 주차면 ID | Long |  | X | 10 |
| data.spotNumber | 주차면 번호 | String |  | X | "C-01" |
| data.spotType | 주차면 유형 | String |  | X | "NORMAL" |
| data.isOccupied | 점유 여부 | Boolean |  | X | false |
| data.positionX | 지도상 X 좌표 | Integer |  | O | 200 |
| data.positionY | 지도상 Y 좌표 | Integer |  | O | 300 |
| data.zoneId | 주차장 구역 ID | Long |  | X | 5 |
| data.zoneName | 주차장 구역명 | String |  | X | "지하 주차장" |
| data.deviceId | 초음파 센서 장치 ID | Long |  | O | 20 |
| data.deviceName | 초음파 센서 장치명 | String |  | O | "C-01 초음파 센서" |
| data.status | 주차면 상태 | String |  | X | "ACTIVE" |
| data.createdAt | 등록일시 | String |  | X | "2026-04-01T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "주차면이 등록되었습니다.",
  "data": {
    "id": 10,
    "spotNumber": "C-01",
    "spotType": "NORMAL",
    "isOccupied": false,
    "positionX": 200,
    "positionY": 300,
    "zoneId": 5,
    "zoneName": "지하 주차장",
    "deviceId": 20,
    "deviceName": "C-01 초음파 센서",
    "status": "ACTIVE",
    "createdAt": "2026-04-01T09:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 201 | 주차면 등록 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 ID 또는 장치 ID |
| 409 | 동일 구역 내 중복 주차면 번호 |
| 409 | 이미 다른 주차면에 연결된 장치 |
