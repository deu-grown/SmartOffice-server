# 급여 기준 등록

카테고리: 급여
설명: 직급별 급여 기준을 등록한다. 동일 직급의 기존 기준이 있으면 effectiveTo를 자동으로 설정하고 새 기준을 등록한다.
Method: POST
URL: /api/v1/salary/settings
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| position | 직급명 | String |  | X | "사원" |
| baseSalary | 기본급 (원) | Integer |  | X | 2500000 |
| overtimeRate | 초과근무 배율 | Double |  | X | 1.5 |
| nightRate | 야간수당 배율 | Double |  | O | 2.0 |
| effectiveFrom | 적용 시작일 | String |  | X | "2026-01-01" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "position": "사원",
  "baseSalary": 2500000,
  "overtimeRate": 1.5,
  "nightRate": 2.0,
  "effectiveFrom": "2026-01-01"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "급여 기준이 등록되었습니다." |
| data.id | 생성된 급여 기준 ID | Long |  | X | 1 |
| data.position | 직급명 | String |  | X | "사원" |
| data.baseSalary | 기본급 (원) | Integer |  | X | 2500000 |
| data.overtimeRate | 초과근무 배율 | Double |  | X | 1.5 |
| data.nightRate | 야간수당 배율 | Double |  | X | 2.0 |
| data.effectiveFrom | 적용 시작일 | String |  | X | "2026-01-01" |
| data.effectiveTo | 적용 종료일 | String |  | O | null |
| data.createdAt | 등록일시 | String |  | X | "2026-03-02T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "급여 기준이 등록되었습니다.",
  "data": {
    "id": 1,
    "position": "사원",
    "baseSalary": 2500000,
    "overtimeRate": 1.5,
    "nightRate": 2.0,
    "effectiveFrom": "2026-01-01",
    "effectiveTo": null,
    "createdAt": "2026-03-02T09:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 201 | 급여 기준 등록 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 409 | 동일 직급의 적용 시작일이 중복됨 |
