# 내 급여 내역 조회

카테고리: 급여
설명: 로그인한 직원 본인의 급여 내역을 조회한다. CONFIRMED 상태인 내역만 반환한다.
Method: GET
URL: /api/v1/salary/records/me
param: year, month
담당자: 박성종
사용자: USER
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| year | 년도 필터 | Integer | O | 2026 |
| month | 월 필터 | Integer | O | 4 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 급여 내역 ID | Long |  | X | 1 |
| data[].year | 년도 | Integer |  | X | 2026 |
| data[].month | 월 | Integer |  | X | 4 |
| data[].baseSalary | 기본급 (원) | Integer |  | X | 2500000 |
| data[].overtimePay | 초과근무 수당 (원) | Integer |  | X | 150000 |
| data[].totalPay | 최종 지급액 (원) | Integer |  | X | 2650000 |
| data[].totalWorkMinutes | 총 근무시간 (분) | Integer |  | X | 10080 |
| data[].overtimeMinutes | 초과근무 시간 (분) | Integer |  | X | 120 |
| data[].status | 급여 상태 | String |  | X | "CONFIRMED" |
| data[].createdAt | 산출일시 | String |  | X | "2026-05-01T02:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": [
    {
      "id": 1,
      "year": 2026,
      "month": 4,
      "baseSalary": 2500000,
      "overtimePay": 150000,
      "totalPay": 2650000,
      "totalWorkMinutes": 10080,
      "overtimeMinutes": 120,
      "status": "CONFIRMED",
      "createdAt": "2026-05-01T02:00:00"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 내 급여 내역 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
