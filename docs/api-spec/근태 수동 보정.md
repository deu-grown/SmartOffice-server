# 근태 수동 보정

카테고리: 근태
설명: 근태를 수정합니다.
Method: PUT
URL: /api/v1/attendance/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 5

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 근태 여부 | string | 필수 | X | "출근” |
| checkInTime | 출근 시간 | string | 필수 | O | "2026-04-01T09:00:00” |
| checkOutTIme | 퇴근 시간 | string | 필수 | O | "2026-04-01T18:00:00” |
| reason | 설명 | string | 선택 | O | “단말기 오류로 인한 수동 입력” |

**Query parameter**

/api/v1/attendance/{id}?status=출근&checkInTime=2026-04-01T09:00:00&checkOutTime=2026-04-01T18:00:00&reason=단말기 오류로 인한 수동 입력

**Example**

```jsx
{
  "status": "출근",
  "checkInTime": "2026-04-01T09:00:00",
  "checkOutTime": "2026-04-01T18:00:00",
  "reason": "단말기 오류로 인한 수동 입력"
}
```
