# 자산 이탈 알림 목록 조회

카테고리: 자산관리
설명: 자산 이탈 감지 알림 목록을 조회한다. 미해제 알림만 조회하거나 전체 이력 조회 가능.
Method: GET
URL: /api/v1/assets/alerts
param: assetId, resolved, startDate, endDate
담당자: 박성종
사용자: ADMIN
우선순위: 8

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| assetId | 자산 ID 필터 | Long | O | 1 |
| resolved | 해제 여부 필터 (false=미해제만) | Boolean | O | false |
| startDate | 조회 시작일 | String | O | "2026-04-01" |
| endDate | 조회 종료일 | String | O | "2026-04-30" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 알림 ID | Long |  | X | 5 |
| data[].assetId | 자산 ID | Long |  | X | 1 |
| data[].assetName | 자산명 | String |  | X | "맥북 프로 16인치" |
| data[].assetNumber | 자산 번호 | String |  | X | "ASSET-001" |
| data[].alertType | 알림 유형 | String |  | X | "ZONE_VIOLATION" |
| data[].detectedDeviceId | 감지 장치 ID | Long |  | X | 10 |
| data[].detectedDeviceName | 감지 장치명 | String |  | X | "1층 서버실 RFID 리더기" |
| data[].detectedZoneId | 감지된 구역 ID | Long |  | X | 3 |
| data[].detectedZoneName | 감지된 구역명 | String |  | X | "1층 서버실" |
| data[].allowedZoneId | 허용 구역 ID | Long |  | X | 2 |
| data[].allowedZoneName | 허용 구역명 | String |  | X | "2층 개발팀" |
| data[].resolvedAt | 해제 일시 | String |  | O | null |
| data[].createdAt | 감지 일시 | String |  | X | "2026-04-01T14:32:00" |

### Status

| status | response content |
| --- | --- |
| 200 | 자산 이탈 알림 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
