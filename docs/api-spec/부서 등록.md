# 부서 등록

카테고리: 부서
설명: 새로운 부서를 등록한다.
Method: POST
URL: /api/v1/departments
담당자: 박성종
사용자: ADMIN
우선순위: 3

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 부서명 | String |  | X | "개발팀" |
| description | 부서 설명 | String |  | O | "백엔드 개발 담당" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "name": "개발팀",
  "description": "백엔드 개발 담당"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "부서가 등록되었습니다." |
| data.id | 생성된 부서 ID | Long |  | X | 1 |
| data.name | 부서명 | String |  | X | "개발팀" |
| data.description | 부서 설명 | String |  | O | "백엔드 개발 담당" |
| data.createdAt | 생성일시 | String |  | X | "2026-03-02T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "부서가 등록되었습니다.",
  "data": {
    "id": 1,
    "name": "개발팀",
    "description": "백엔드 개발 담당",
    "createdAt": "2026-03-02T09:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 201 | 부서 등록 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 409 | 이미 존재하는 부서명 |
