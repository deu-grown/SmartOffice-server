# 주차장 전체 현황 조회

카테고리: 주차
설명: 특정 주차장 구역의 주차면 목록과 점유 현황을 조회한다. 주차면 유형, 점유 여부로 필터링 가능.
Method: GET
URL: /api/v1/parking/zones/{zoneId}/spots
param: spotType, isOccupied
담당자: 우하
사용자: ALL
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| spotType | 주차면 유형 필터 | String | O | "DISABLED" |
| isOccupied | 점유 여부 필터 | Boolean | O | false |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.zoneId | 주차장 구역 ID | Long |  | X | 5 |
| data.zoneName | 주차장 구역명 | String |  | X | "지하 주차장" |
| data.totalSpots | 전체 주차면 수 | Integer |  | X | 30 |
| data.occupiedSpots | 점유 주차면 수 | Integer |  | X | 12 |
| data.availableSpots | 빈 주차면 수 | Integer |  | X | 18 |
| data.spots[].id | 주차면 ID | Long |  | X | 1 |
| data.spots[].spotNumber | 주차면 번호 | String |  | X | "A-01" |
| data.spots[].spotType | 주차면 유형 | String |  | X | "NORMAL" |
| data.spots[].isOccupied | 점유 여부 | Boolean |  | X | false |
| data.spots[].deviceId | 초음파 센서 장치 ID | Long |  | O | 15 |
| data.spots[].deviceName | 초음파 센서 장치명 | String |  | O | "A-01 초음파 센서" |
| data.spots[].status | 주차면 상태 | String |  | X | "ACTIVE" |

### Status

| status | response content |
| --- | --- |
| 200 | 주차장 현황 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 404 | 존재하지 않는 구역 |
