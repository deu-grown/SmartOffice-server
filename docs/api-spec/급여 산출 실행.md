# 급여 산출 실행

카테고리: 급여
설명: 특정 연월의 급여를 산출한다. monthly_attendance 기준으로 해당 직급의 salary_settings를 적용하여 salary_records를 생성한다. 이미 CONFIRMED 상태인 내역은 재산출 불가.
Method: POST
URL: /api/v1/salary/records/calculate
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| year | 산출 년도 | Integer |  | X | 2026 |
| month | 산출 월 | Integer |  | X | 4 |
| userIds | 산출 대상 직원 ID 목록 (비어있으면 전체) | Array |  | O | [1, 2, 3] |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "year": 2026,
  "month": 4,
  "userIds": [1, 2, 3]
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "급여 산출이 완료되었습니다." |
| data.year | 산출 년도 | Integer |  | X | 2026 |
| data.month | 산출 월 | Integer |  | X | 4 |
| data.totalCount | 산출 대상 총 인원 | Integer |  | X | 3 |
| data.successCount | 산출 성공 인원 | Integer |  | X | 3 |
| data.skipCount | 건너뜀 인원 (CONFIRMED 등) | Integer |  | X | 0 |
| data.records[].userId | 직원 ID | Long |  | X | 1 |
| data.records[].userName | 직원 이름 | String |  | X | "박성종" |
| data.records[].totalPay | 최종 지급액 (원) | Integer |  | X | 2650000 |
| data.records[].status | 급여 상태 | String |  | X | "DRAFT" |

**Example**

```json
{
  "code": "success",
  "message": "급여 산출이 완료되었습니다.",
  "data": {
    "year": 2026,
    "month": 4,
    "totalCount": 3,
    "successCount": 3,
    "skipCount": 0,
    "records": [
      {
        "userId": 1,
        "userName": "박성종",
        "totalPay": 2650000,
        "status": "DRAFT"
      },
      {
        "userId": 2,
        "userName": "김동우",
        "totalPay": 2500000,
        "status": "DRAFT"
      },
      {
        "userId": 3,
        "userName": "류경호",
        "totalPay": 2520000,
        "status": "DRAFT"
      }
    ]
  }

```

### Status

| status | response content |
| --- | --- |
| 200 | 급여 산출 완료 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 해당 월 근태 집계 데이터 없음 |
| 409 | 해당 직급의 적용 가능한 급여 기준 없음 |
