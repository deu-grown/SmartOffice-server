# 내 정보 수정

카테고리: 직원/인사관리
설명: 로그인한 직원 본인이 수정 가능한 정보를 변경한다. 권한·직급·부서는 수정 불가.
Method: POST
URL: /api/v1/users/me
담당자: 박성종
사용자: USER
우선순위: 2

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| phone | 연락처 | String |  | O | "010-1111-2222" |
| password | 새 비밀번호 | String |  | O | "newPassword123!" |
| currentPassword | 현재 비밀번호 (비밀번호 변경 시 필수) | String |  | O | "password123!" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "phone": "010-1111-2222",
  "password": "newPassword123!",
  "currentPassword": "password123!"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정보가 수정되었습니다." |
| data.phone | 연락처 | String |  | O | "010-1111-2222" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정보가 수정되었습니다.",
  "data": {
    "phone": "010-1111-2222",
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 내 정보 수정 성공 |
| 400 | 요청 값 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 422 | 현재 비밀번호 불일치 |
