# 내 정보 조회

카테고리: 인증
설명: 내 정보를 조회합니다.
Method: GET
URL: /api/v1/auth/me
담당자: 박성종
사용자: ALL
우선순위: 1

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.id | 사용자 ID | Long |  | X | 1 |
| data.employeeNumber | 사번 | String |  | X | "EMP001" |
| data.name | 이름 | String |  | X | "박성종" |
| data.email | 이메일 | String |  | X | "admin@grown.com" |
| data.role | 권한 | String |  | X | "ADMIN" |
| data.position | 직급 | String |  | X | "팀장" |
| data.department | 부서명 | String |  | O | "개발팀" |
| data.phone | 연락처 | String |  | O | "010-1234-5678" |
| data.hiredAt | 입사일 | String |  | O | "2024-03-02" |
| data.status | 재직 상태 | String |  | X | "ACTIVE" |

### Status

| status | response content |
| --- | --- |
| 200 | 사용자 정보 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 퇴사 처리된 계정 |
