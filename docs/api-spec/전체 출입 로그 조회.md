# 전체 출입 로그 조회

카테고리: 출입 로그
설명: 전체 출입 로그를 조회한다. 구역, 직원, 인증 결과, 날짜 범위로 필터링 가능. 최신순 정렬.
Method: GET
URL: /api/v1/access-logs
param: zoneId, userId, authResult, direction, startDate, endDate, page, size
담당자: 우하
사용자: ADMIN
우선순위: 5

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| zoneId | 구역 ID 필터 | Long | O | 1 |
| userId | 직원 ID 필터 | Long | O | 1 |
| authResult | 인증 결과 필터 | String | O | "DENIED" |
| direction | 방향 필터 | String | O | "IN" |
| startDate | 조회 시작일 | String | O | "2026-04-01" |
| endDate | 조회 종료일 | String | O | "2026-04-30" |
| page | 페이지 번호 (0부터 시작) | Integer | O | 0 |
| size | 페이지 크기 | Integer | O | 20 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.totalElements | 전체 로그 수 | Long |  | X | 1024 |
| data.totalPages | 전체 페이지 수 | Integer |  | X | 52 |
| data.currentPage | 현재 페이지 | Integer |  | X | 0 |
| data.logs[].id | 로그 ID | Long |  | X | 1024 |
| data.logs[].userId | 직원 ID | Long |  | O | 1 |
| data.logs[].userName | 직원 이름 | String |  | O | "박성종" |
| data.logs[].employeeNumber | 사번 | String |  | O | "EMP001" |
| data.logs[].uid | 태그된 UID | String |  | X | "04:A3:2F:1B" |
| data.logs[].deviceId | 장치 ID | Long |  | X | 1 |
| data.logs[].deviceName | 장치명 | String |  | X | "1층 입구 NFC 리더기" |
| data.logs[].zoneId | 구역 ID | Long |  | X | 1 |
| data.logs[].zoneName | 구역명 | String |  | X | "1층" |
| data.logs[].direction | 입실/퇴실 | String |  | X | "IN" |
| data.logs[].authResult | 인증 결과 | String |  | X | "APPROVED" |
| data.logs[].denyReason | 거부 사유 | String |  | O | null |
| data.logs[].taggedAt | 태그 일시 | String |  | X | "2026-04-01T09:02:11" |

### Status

| status | response content |
| --- | --- |
| 200 | 출입 로그 조회 성공 |
| 400 | 날짜 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
