# 부서 목록 조회

카테고리: 부서
설명: 전체 부서 목록을 조회한다.
Method: GET
URL: /api/v1/departments
담당자: 박성종
사용자: ADMIN
우선순위: 3

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 부서 ID | Long |  | X | 1 |
| data[].name | 부서명 | String |  | X | "개발팀" |
| data[].description | 부서 설명 | String |  | O | "백엔드 개발 담당" |
| data[].userCount | 소속 인원 수 | Integer |  | X | 5 |
| data[].createdAt | 생성일시 | String |  | X | "2026-03-02T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": [
    {
      "id": 1,
      "name": "개발팀",
      "description": "백엔드 개발 담당",
      "userCount": 5,
      "createdAt": "2026-03-02T09:00:00"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 부서 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
