# 요금 산출 수동 실행

카테고리: 전력관리
설명: 특정 연월의 전력 요금을 수동으로 산출한다. 매월 1일 자동 실행되는 스케줄러와 동일한 로직으로 동작하며, 이미 산출된 내역이 있으면 덮어쓴다.
Method: POST
URL: /api/v1/power/billing/calculate
담당자: 박성종
사용자: ADMIN
우선순위: 10

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| year | 산출 년도 | Integer |  | X | 2026 |
| month | 산출 월 | Integer |  | X | 4 |
| unitPrice | 적용 단가 (원/kWh) | Double |  | X | 120.5 |
| baseFee | 기본요금 (원) | Integer |  | X | 6160 |
| zoneIds | 산출 대상 구역 ID 목록 (비어있으면 전체) | Array |  | O | [1, 2, 3] |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "year": 2026,
  "month": 4,
  "unitPrice": 120.5,
  "baseFee": 6160,
  "zoneIds": [1, 2, 3]
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "전력 요금 산출이 완료되었습니다." |
| data.year | 산출 년도 | Integer |  | X | 2026 |
| data.month | 산출 월 | Integer |  | X | 4 |
| data.totalCount | 산출 대상 총 구역 수 | Integer |  | X | 3 |
| data.successCount | 산출 성공 구역 수 | Integer |  | X | 3 |
| data.skipCount | 건너뜀 구역 수 (데이터 없음) | Integer |  | X | 0 |
| data.totalFee | 전체 합계 요금 (원) | Integer |  | X | 117340 |

**Example**

```json
{
  "code": "success",
  "message": "전력 요금 산출이 완료되었습니다.",
  "data": {
    "year": 2026,
    "month": 4,
    "totalCount": 3,
    "successCount": 3,
    "skipCount": 0,
    "totalFee": 117340
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 전력 요금 산출 완료 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 해당 월 전력 집계 데이터 없음 |
