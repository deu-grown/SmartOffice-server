# 전체 구역 월 요금 현황 조회

카테고리: 전력관리
설명: 전체 구역의 특정 월 전력 요금 현황을 조회한다. 구역별 사용량 및 요금 비교에 사용한다.
Method: GET
URL: /api/v1/power/billing
param: year, month
담당자: 박성종
사용자: ADMIN
우선순위: 10

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| year | 년도 | Integer | O | 2026 |
| month | 월 | Integer | O | 4 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.year | 년도 | Integer |  | X | 2026 |
| data.month | 월 | Integer |  | X | 4 |
| data.totalKwh | 전체 구역 총 사용량 (kWh) | Double |  | X | 924.81 |
| data.totalFee | 전체 구역 총 요금 (원) | Integer |  | X | 117340 |
| data.zones[].zoneId | 구역 ID | Long |  | X | 1 |
| data.zones[].zoneName | 구역명 | String |  | X | "1층" |
| data.zones[].totalKwh | 구역 총 사용량 (kWh) | Double |  | X | 183.24 |
| data.zones[].totalFee | 구역 총 요금 (원) | Integer |  | X | 28240 |

### Status

| status | response content |
| --- | --- |
| 200 | 전체 구역 월 요금 현황 조회 성공 |
| 400 | 날짜 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
