# 전체 급여 내역 조회

카테고리: 급여
설명: 전체 직원의 급여 내역을 조회한다. 연월, 직원, 상태로 필터링 가능.
Method: GET
URL: /api/v1/salary/records
param: year, month, userId, status
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| year | 년도 필터 | Integer | O | 2026 |
| month | 월 필터 | Integer | O | 4 |
| userId | 직원 ID 필터 | Long | O | 1 |
| status | 급여 상태 필터 | String | O | "CONFIRMED" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 급여 내역 ID | Long |  | X | 1 |
| data[].userId | 직원 ID | Long |  | X | 1 |
| data[].userName | 직원 이름 | String |  | X | "박성종" |
| data[].employeeNumber | 사번 | String |  | X | "EMP001" |
| data[].position | 직급 | String |  | X | "팀장" |
| data[].department | 부서명 | String |  | O | "개발팀" |
| data[].year | 년도 | Integer |  | X | 2026 |
| data[].month | 월 | Integer |  | X | 4 |
| data[].baseSalary | 기본급 (원) | Integer |  | X | 2500000 |
| data[].overtimePay | 초과근무 수당 (원) | Integer |  | X | 150000 |
| data[].totalPay | 최종 지급액 (원) | Integer |  | X | 2650000 |
| data[].status | 급여 상태 | String |  | X | "CONFIRMED" |
| data[].createdAt | 산출일시 | String |  | X | "2026-05-01T02:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",

```

### Status

| status | response content |
| --- | --- |
| 200 | 전체 급여 내역 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
