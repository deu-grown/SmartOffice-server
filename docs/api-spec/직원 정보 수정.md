# 직원 정보 수정

카테고리: 직원/인사관리
설명: 특정 직원의 정보를 수정한다.
Method: PUT
URL: /api/v1/users/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 2

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 이름 | String |  | O | "김동우" |
| role | 권한 | String |  | O | "ADMIN" |
| position | 직급 | String |  | O | "주임" |
| departmentId | 부서 ID | Long |  | O | 2 |
| phone | 연락처 | String |  | O | "010-9876-5432" |
| hiredAt | 입사일 | String |  | O | "2026-03-02" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "name": "김동우",
  "role": "ADMIN",
  "position": "주임",
  "departmentId": 2,
  "phone": "010-9876-5432",
  "hiredAt": "2026-03-02"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "직원 정보가 수정되었습니다." |
| data.id | 직원 ID | Long |  | X | 2 |
| data.name | 이름 | String |  | X | "김동우" |
| data.role | 권한 | String |  | X | "ADMIN" |
| data.position | 직급 | String |  | X | "주임" |
| data.department | 부서명 | String |  | O | "IoT팀" |
| data.phone | 연락처 | String |  | O | "010-9876-5432" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "직원 정보가 수정되었습니다.",
  "data": {
    "id": 2,
    "name": "김동우",
    "role": "ADMIN",
    "position": "주임",
    "department": "IoT팀",
    "phone": "010-9876-5432",
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 직원 정보 수정 성공 |
| 400 | 요청 값 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 직원 또는 부서 |
