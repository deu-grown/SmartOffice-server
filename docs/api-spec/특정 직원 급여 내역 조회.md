# 특정 직원 급여 내역 조회

카테고리: 급여
설명: 특정 직원의 전체 급여 내역을 조회한다. 연월로 필터링 가능.
Method: GET
URL: /api/v1/users/{id}/salary/records
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| year | 년도 필터 | Integer | O | 2026 |
| month | 월 필터 | Integer | O | 4 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.userId | 직원 ID | Long |  | X | 1 |
| data.userName | 직원 이름 | String |  | X | "박성종" |
| data.records[].id | 급여 내역 ID | Long |  | X | 1 |
| data.records[].year | 년도 | Integer |  | X | 2026 |
| data.records[].month | 월 | Integer |  | X | 4 |
| data.records[].baseSalary | 기본급 (원) | Integer |  | X | 2500000 |
| data.records[].overtimePay | 초과근무 수당 (원) | Integer |  | X | 150000 |
| data.records[].totalPay | 최종 지급액 (원) | Integer |  | X | 2650000 |
| data.records[].totalWorkMinutes | 총 근무시간 (분) | Integer |  | X | 10080 |
| data.records[].overtimeMinutes | 초과근무 시간 (분) | Integer |  | X | 120 |
| data.records[].status | 급여 상태 | String |  | X | "CONFIRMED" |
| data.records[].createdAt | 산출일시 | String |  | X | "2026-05-01T02:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": {
    "userId": 1,
    "userName": "박성종",
    "records": [
      {
        "id": 1,
        "year": 2026,
        "month": 4,
        "baseSalary": 2500000,
        "overtimePay": 150000,
        "totalPay": 2650000,
        "totalWorkMinutes": 10080,
        "overtimeMinutes": 120,
        "status": "CONFIRMED",
        "createdAt": "2026-05-01T02:00:00"
      }
    ]
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 특정 직원 급여 내역 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 직원 |
