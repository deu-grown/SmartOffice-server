# 주차면 수정

카테고리: 주차
설명: 주차면 정보를 수정한다. 센서 교체, 좌표 조정, 주차면 유형 변경 등에 사용한다.
Method: PUT
URL: /api/v1/parking/spots/{id}
담당자: 우하
사용자: ADMIN
우선순위: 6

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| spotNumber | 주차면 번호 | String |  | O | "C-01" |
| spotType | 주차면 유형 | String |  | O | "EV" |
| deviceId | 초음파 센서 장치 ID | Long |  | O | 21 |
| positionX | 지도상 X 좌표 | Integer |  | O | 210 |
| positionY | 지도상 Y 좌표 | Integer |  | O | 310 |
| status | 주차면 상태 | String |  | O | "INACTIVE" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "spotType": "EV",
  "deviceId": 21,
  "positionX": 210,
  "positionY": 310,
  "status": "ACTIVE"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "주차면 정보가 수정되었습니다." |
| data.id | 주차면 ID | Long |  | X | 10 |
| data.spotNumber | 주차면 번호 | String |  | X | "C-01" |
| data.spotType | 주차면 유형 | String |  | X | "EV" |
| data.positionX | 지도상 X 좌표 | Integer |  | O | 210 |
| data.positionY | 지도상 Y 좌표 | Integer |  | O | 310 |
| data.deviceId | 초음파 센서 장치 ID | Long |  | O | 21 |
| data.deviceName | 초음파 센서 장치명 | String |  | O | "C-01 초음파 센서 (교체)" |
| data.status | 주차면 상태 | String |  | X | "ACTIVE" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "주차면 정보가 수정되었습니다.",
  "data": {
    "id": 10,
    "spotNumber": "C-01",
    "spotType": "EV",
    "positionX": 210,
    "positionY": 310,
    "deviceId": 21,
    "deviceName": "C-01 초음파 센서 (교체)",
    "status": "ACTIVE",
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 주차면 수정 성공 |
| 400 | 요청 값 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 주차면 또는 장치 ID |
| 409 | 동일 구역 내 중복 주차면 번호 |
| 409 | 이미 다른 주차면에 연결된 장치 |
