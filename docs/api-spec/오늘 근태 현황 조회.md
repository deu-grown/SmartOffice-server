# 오늘 근태 현황 조회

카테고리: 대시보드
설명: 오늘 날짜 기준 전체 직원의 근태 현황을 조회한다. 부서별 필터링 가능.
Method: GET
URL: /api/v1/dashboard/attendance/today
담당자: 박성종
사용자: ADMIN
우선순위: 7

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | 관리자 인증 토큰 (Bearer) | Bearer eyJhbGciOiJIUzI1Ni... |
| Content-Type | 데이터 전송 형식 | application/json |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 응답 상태 | String | 필수 | No | "success" |
| message | 결과 메시지 | String | 필수 | No | "근태 현황 조회가 완료되었습니다." |
| data.presentCount | 정상 출근 완료 인원 | Integer | 필수 | No | 45 |
| data.absentCount | 아직 출근하지 않은 인원 | Integer | 필수 | No | 5 |
| data.lateCount | 정해진 시간보다 늦게 체크인한 인원 | Integer | 필수 | No | 2 |
| data.totalExpect | 오늘 출근이 예정된 총 인원 | Integer | 필수 | No | 52 |

**Example**

```json
{
  "status": "success",
  "message": "근태 현황 조회가 완료되었습니다.",
  "data": {
    "presentCount": 45,
    "absentCount": 5,
    "lateCount": 2,
    "totalExpect": 52
  }
}
```

### Status

| status | response content | 설명 |
| --- | --- | --- |
| 200 | OK | 조회 성공 |
| 400 | Bad Request | 날짜 형식이 잘못됨 (YYYY-MM-DD 아님) |
| 401 | Unauthorized | 관리자 인증 실패 |
