# 구역 삭제

카테고리: 구역
설명: 구역을 삭제한다. 하위 구역이나 설치된 장치가 존재하는 경우 삭제 불가.
Method: DELETE
URL: /api/v1/zones/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 4

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "구역이 삭제되었습니다." |

**Example**

```json
{
  "code": "success",
  "message": "구역이 삭제되었습니다."
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 구역 삭제 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 |
| 409 | 하위 구역이 존재하여 삭제 불가 |
| 409 | 설치된 장치가 존재하여 삭제 불가 |
