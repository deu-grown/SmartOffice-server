# 구역 목록 조회

카테고리: 구역
설명: 전체 구역을 평면 리스트로 조회한다. 구역 유형으로 필터링 가능.
Method: GET
URL: /api/v1/zones
param: zoneType, parentId
담당자: 박성종
사용자: ADMIN
우선순위: 4

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| zoneType | 구역 유형 필터 | String | O | "FLOOR" |
| parentId | 상위 구역 ID 필터 | Long | O | 1 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 구역 ID | Long |  | X | 1 |
| data[].name | 구역명 | String |  | X | "1층" |
| data[].zoneType | 구역 유형 | String |  | X | "FLOOR" |
| data[].description | 구역 설명 | String |  | O | "1층 전체 구역" |
| data[].parentId | 상위 구역 ID | Long |  | O | null |
| data[].parentName | 상위 구역명 | String |  | O | null |
| data[].deviceCount | 설치 장치 수 | Integer |  | X | 3 |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": [
    {
      "id": 1,

      "zoneType": "FLOOR",
      "description": "1층 전체 구역",

    },
    {
      "id": 2,

    }
  ]

```

### Status

| status | response content |
| --- | --- |
| 200 | 구역 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
