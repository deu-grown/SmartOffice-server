# 주차면 목록 조회 (관리)

카테고리: 주차
설명: 전체 주차면 목록을 조회한다. 관리자 전용 관리 화면에서 사용한다.
Method: GET
URL: /api/v1/parking/spots
param: zoneId, spotType, status
담당자: 우하
사용자: ADMIN
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| zoneId | 주차장 구역 ID 필터 | Long | O | 5 |
| spotType | 주차면 유형 필터 | String | O | "EV" |
| status | 주차면 상태 필터 | String | O | "ACTIVE" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 주차면 ID | Long |  | X | 1 |
| data[].spotNumber | 주차면 번호 | String |  | X | "A-01" |
| data[].spotType | 주차면 유형 | String |  | X | "NORMAL" |
| data[].isOccupied | 점유 여부 | Boolean |  | X | false |
| data[].positionX | 지도상 X 좌표 | Integer |  | O | 100 |
| data[].positionY | 지도상 Y 좌표 | Integer |  | O | 200 |
| data[].zoneId | 주차장 구역 ID | Long |  | X | 5 |
| data[].zoneName | 주차장 구역명 | String |  | X | "지하 주차장" |
| data[].deviceId | 초음파 센서 장치 ID | Long |  | O | 15 |
| data[].deviceName | 초음파 센서 장치명 | String |  | O | "A-01 초음파 센서" |
| data[].status | 주차면 상태 | String |  | X | "ACTIVE" |
| data[].createdAt | 등록일시 | String |  | X | "2026-03-02T09:00:00" |

### Status

| status | response content |
| --- | --- |
| 200 | 주차면 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
