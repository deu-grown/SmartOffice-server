# 구역별 시간별 전력 이력 조회

카테고리: 전력관리
설명: 특정 구역의 시간별 전력 집계 이력을 조회한다. 날짜 범위로 필터링 가능.
Method: GET
URL: /api/v1/power/zones/{zoneId}/hourly
param: startDate, endDate, deviceId
담당자: 박성종
사용자: ADMIN
우선순위: 10

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| startDate | 조회 시작일 | String | O | "2026-04-01" |
| endDate | 조회 종료일 | String | O | "2026-04-30" |
| deviceId | 장치 ID 필터 | Long | O | 5 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.zoneId | 구역 ID | Long |  | X | 1 |
| data.zoneName | 구역명 | String |  | X | "1층" |
| data.logs[].id | 집계 로그 ID | Long |  | X | 1 |
| data.logs[].deviceId | 장치 ID | Long |  | X | 5 |
| data.logs[].deviceName | 장치명 | String |  | X | "1층 전력 센서" |
| data.logs[].hourAt | 집계 시각 (정각 기준) | String |  | X | "2026-04-01T13:00:00" |
| data.logs[].kwh | 시간당 사용량 (kWh) | Double |  | X | 0.2457 |
| data.logs[].avgWatt | 평균 전력 (W) | Double |  | X | 245.7 |
| data.logs[].peakWatt | 최대 순간 전력 (W) | Double |  | X | 312.4 |

### Status

| status | response content |
| --- | --- |
| 200 | 시간별 전력 이력 조회 성공 |
| 400 | 날짜 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 |
