# 자산 이탈 알림 해제

카테고리: 자산관리
설명: 자산 이탈 알림을 해제 처리한다. resolvedAt에 현재 일시를 기록한다.
Method: PUT
URL: /api/v1/assets/alerts/{id}/resolve
담당자: 박성종
사용자: ADMIN
우선순위: 8

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "알림이 해제되었습니다." |
| data.id | 알림 ID | Long |  | X | 5 |
| data.assetId | 자산 ID | Long |  | X | 1 |
| data.assetName | 자산명 | String |  | X | "맥북 프로 16인치" |
| data.resolvedAt | 해제 일시 | String |  | X | "2026-04-01T15:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "알림이 해제되었습니다.",
  "data": {
    "id": 5,
    "assetId": 1,
    "assetName": "맥북 프로 16인치",
    "resolvedAt": "2026-04-01T15:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 알림 해제 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 알림 |
| 409 | 이미 해제된 알림 |
