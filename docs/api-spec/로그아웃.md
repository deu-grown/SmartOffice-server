# 로그아웃

카테고리: 인증
설명: Refresh Token을 서버에서 삭제하고 로그아웃 처리한다.
Method: POST
URL: /api/v1/auth/logout
담당자: 박성종
사용자: ALL
우선순위: 1

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| refreshToken | 발급받은 Refresh Token | String | X | X | "eyJhbGci..." |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "로그아웃 되었습니다." |

**Example**

```json
{
  "code": "success",
  "message": "로그아웃 되었습니다."
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 로그아웃 성공, 서버 Refresh Token 삭제 |
| 400 | refreshToken 누락 |
| 401 | 유효하지 않은 Access Token |
