# 장치 정보 수정

카테고리: 장치
설명: 장치 정보 및 상태를 수정한다. 구역 이동 시 MQTT 토픽이 자동으로 재생성된다.
Method: PUT
URL: /api/v1/devices/{id}
담당자: 우하
사용자: ADMIN
우선순위: 1

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 장치명 | String |  | O | "1층 입구 NFC 리더기 (교체)" |
| zoneId | 설치 구역 ID | Long |  | O | 2 |
| serialNumber | 시리얼 번호 | String |  | O | "SN-NFC-002" |
| status | 장치 상태 | String |  | O | "INACTIVE" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "name": "1층 입구 NFC 리더기 (교체)",
  "zoneId": 2,
  "serialNumber": "SN-NFC-002",
  "status": "INACTIVE"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "장치 정보가 수정되었습니다." |
| data.id | 장치 ID | Long |  | X | 1 |
| data.name | 장치명 | String |  | X | "1층 입구 NFC 리더기 (교체)" |
| data.deviceType | 장치 유형 | String |  | X | "NFC_READER" |
| data.serialNumber | 시리얼 번호 | String |  | O | "SN-NFC-002" |
| data.mqttTopic | 재생성된 MQTT 토픽 | String |  | X | "smartoffice/zones/2/devices/1/nfc" |
| data.zoneId | 설치 구역 ID | Long |  | X | 2 |
| data.zoneName | 설치 구역명 | String |  | X | "1층 회의실 A" |
| data.status | 장치 상태 | String |  | X | "INACTIVE" |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "장치 정보가 수정되었습니다.",
  "data": {
    "id": 1,
    "name": "1층 입구 NFC 리더기 (교체)",
    "deviceType": "NFC_READER",
    "serialNumber": "SN-NFC-002",
    "mqttTopic": "smartoffice/zones/2/devices/1/nfc",
    "zoneId": 2,
    "zoneName": "1층 회의실 A",
    "status": "INACTIVE",
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 장치 수정 성공 |
| 400 | 요청 값 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 장치 또는 구역 ID |
| 409 | 이미 등록된 시리얼 번호 |
