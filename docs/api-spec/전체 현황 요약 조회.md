# 전체 현황 요약 조회

카테고리: 대시보드
설명: 관리자 대시보드 메인에 표시할 전체 현황 요약 데이터를 조회한다. 오늘 근태, 현재 환경, 최근 출입, 주차 현황 등 핵심 지표를 한번에 반환한다.
Method: GET
URL: /api/v1/dashboard/summary
담당자: 박성종
사용자: ADMIN
우선순위: 7

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | 관리자 전용 인증 토큰 (Bearer) | Bearer eyJhbGciOiJIUzI1Ni... |
| Content-Type | 데이터 전송 형식 | application/json |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 응답 상태 | String | 필수 | No | "success" |
| message | 결과 메시지 | String | 필수 | No | "요약 정보 조회가 완료되었습니다." |
| data.totalUsers | 시스템에 등록된 총 사용자 수 | Integer | 필수 | No | 150 |
| data.todayReservations | 금일 전체 예약 신청 건수 | Integer | 필수 | No | 12 |
| data.activeDevices | 현재 정상 가동 중인 IoT 장치 수 | Integer | 필수 | No | 8 |
| data.pendingApprovals | 승인 대기 중인 예약/가입 건수 | Integer | 필수 | No | 3 |

**Example**

```json
{
  "status": "success",
  "message": "요약 정보 조회가 완료되었습니다.",
  "data": {
    "totalUsers": 150,
    "todayReservations": 12,
    "activeDevices": 8,
    "pendingApprovals": 3
  }
}
```

### Status

| status | response content | 설명 |
| --- | --- | --- |
| 200 | OK | 조회 성공 |
| 401 | Unauthorized | 관리자 인증 실패 |
| 403 | Forbidden | 접근 권한 부족 (일반 유저가 접근 시) |
