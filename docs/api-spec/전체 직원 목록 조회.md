# 전체 직원 목록 조회

카테고리: 직원/인사관리
설명: 전체 직원 목록을 조회한다. 부서, 재직 상태로 필터링 가능.
Method: GET
URL: /api/v1/users
param: departmentId, status, keyword
담당자: 박성종
사용자: ADMIN
우선순위: 2

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| departmentId | 부서 ID 필터 | Long | O | 1 |
| status | 재직 상태 필터 | String | O | "ACTIVE" |
| keyword | 이름/사번 검색 | String | O | "박성종" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 직원 ID | Long |  | X | 1 |
| data[].employeeNumber | 사번 | String |  | X | "EMP001" |
| data[].name | 이름 | String |  | X | "박성종" |
| data[].email | 이메일 | String |  | X | "admin@grown.com" |
| data[].role | 권한 | String |  | X | "ADMIN" |
| data[].position | 직급 | String |  | X | "팀장" |
| data[].department | 부서명 | String |  | O | "개발팀" |
| data[].phone | 연락처 | String |  | O | "010-1234-5678" |
| data[].status | 재직 상태 | String |  | X | "ACTIVE" |
| data[].hiredAt | 입사일 | String |  | O | "2024-03-02" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": [
    {
      "id": 1,
      "employeeNumber": "EMP001",
      "name": "박성종",
      "email": "admin@grown.com",
      "role": "ADMIN",
      "position": "팀장",
      "department": "개발팀",
      "phone": "010-1234-5678",
      "status": "ACTIVE",
      "hiredAt": "2024-03-02"
    }
  ]
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 직원 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
