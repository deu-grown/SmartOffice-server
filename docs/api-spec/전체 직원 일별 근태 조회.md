# 전체 직원 일별 근태 조회

카테고리: 근태
설명: 전체 직원의 일별 근태를 조회합니다.
Method: GET
URL: /api/v1/attendance/daily
담당자: 박성종
사용자: ADMIN
우선순위: 5

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| date | 조회하고자 하는 날짜 | String | 필수 | X | “2026-04-01” |
| name | 조회하고자 하는 직원 이름 | String | 선택 | O | “홍길동” |
| depart | 조회하고자 하는 부서 | String | 선택 | O | “개발부” |

**Query parameter**

/api/v1/attendance/daily&date=2026-04-01&name=홍길동&depart=개발부

**Example (Request Body)**

```jsx
{
	"date": "2026-04-01",
  "name": "홍길동",
  "depart": "개발부"
}
```

| --- | --- | --- | --- | --- | --- |
| status | 상태값 | string |  | X | “success” |
| message | 메시지 | string |  | X | “조회가 완료되었습니다.” |
| data | 데이터 | JSON(object) |  | X | “searchQuery”: {….} |
| searchQuery | 관리자 요청 | JSON(object) |  | X |  "date": "2026-04-01",
      "name": "홍길동",
      "depart": "개발부" |
| totalCount | 검색 결과 | long |  | X | 1 |
| attendanceList | 근태 정보  | array |  |  | {
        "id": 12,
        "name": "홍길동",
        "depart": "개발부",
        "status": "출근",
        "checkInTime": "2026-04-01T08:55:12",
        "checkOutTime": "2026-04-01T18:05:30",
        "workTime": "9시간 10분"
} |
| id | 직원 id | long |  | X | 1 |
| name | 이름 | string |  | X | “홍길동” |
| depart | 부서 | string |  | X | “개발부” |
| atStatus | 근태 여부 | string |  | X | “출근” |
| checkInTime | 출근 시간 | string |  | O | "2026-04-01T08:55:12” |
| checkOutTime | 퇴근 시간 | string |  | O | "2026-04-01T18:05:30” |
| workTime | 근무 시간 | string |  | O | “9시간 10분” |
