# 로그인

카테고리: 인증
설명: 이메일/비밀번호로 로그인하고 JWT 토큰을 발급받는다.
Method: POST
URL: /api/v1/auth/login
담당자: 박성종
사용자: PUBLIC
우선순위: 1

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| email | 로그인 아이디 | String |  | X | "admin@grown.com" |
| password | 로그인 비밀번호 | String |  | X | "password123!” |

**Header**

X

**Query parameter**

X

**Example (Request Body)**

```json
{
  "email": "admin@grown.com",
  "password": "password123!"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "로그인에 성공했습니다." |
| data.accessToken | JWT Access Token | String |  | X | "eyJhbGci..." |
| data.refreshToken | JWT Refresh Token | String |  | X | "eyJhbGci..." |
| data.tokenType | 토큰 타입 | String |  | X | "Bearer" |
| data.expiresIn | Access Token 만료 시간(초) | Integer |  | X | 1800 |
| data.user.id | 사용자 ID | Long |  | X | 1 |
| data.user.name | 사용자 이름 | String |  | X | "박성종" |
| data.user.email | 사용자 이메일 | String |  | X | "admin@grown.com" |
| data.user.role | 권한 | String |  | X | "ADMIN" |
| data.user.position | 직급 | String |  | X | "팀장" |
| data.user.department | 부서명 | String |  | O | "개발팀" |

**Example**

```json
{
  "code": "success",
  "message": "로그인에 성공했습니다.",
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "tokenType": "Bearer",
    "expiresIn": 1800,
    "user": {
      "id": 1,
      "name": "박성종",
      "email": "admin@grown.com",
      "role": "ADMIN",
      "position": "팀장",
      "department": "개발팀"
    }
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 로그인 성공, 토큰 발급 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | 이메일 또는 비밀번호 불일치 |
| 403 | 퇴사 처리된 계정 (status=INACTIVE) |
