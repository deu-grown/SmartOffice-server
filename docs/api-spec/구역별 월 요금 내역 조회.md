# 구역별 월 요금 내역 조회

카테고리: 전력관리
설명: 특정 구역의 월별 전력 요금 산출 내역을 조회한다.
Method: GET
URL: /api/v1/power/zones/{zoneId}/billing
param: year, month
담당자: 박성종
사용자: ADMIN
우선순위: 10

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| year | 년도 필터 | Integer | O | 2026 |
| month | 월 필터 | Integer | O | 4 |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.zoneId | 구역 ID | Long |  | X | 1 |
| data.zoneName | 구역명 | String |  | X | "1층" |
| data.records[].id | 요금 내역 ID | Long |  | X | 1 |
| data.records[].year | 년도 | Integer |  | X | 2026 |
| data.records[].month | 월 | Integer |  | X | 4 |
| data.records[].totalKwh | 월 총 사용량 (kWh) | Double |  | X | 183.24 |
| data.records[].unitPrice | 적용 단가 (원/kWh) | Double |  | X | 120.5 |
| data.records[].baseFee | 기본요금 (원) | Integer |  | X | 6160 |
| data.records[].usageFee | 전력량요금 (원) | Integer |  | X | 22080 |
| data.records[].totalFee | 합계 요금 (원) | Integer |  | X | 28240 |
| data.records[].createdAt | 산출일시 | String |  | X | "2026-05-01T02:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": {
    "zoneId": 1,
    "zoneName": "1층",
    "records": [
      {

        "unitPrice": 120.5,
        "baseFee": 6160,
        "usageFee": 22080,
        "totalFee": 28240,
        "createdAt": "2026-05-01T02:00:00"
      }
    ]
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 구역별 월 요금 내역 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 |
