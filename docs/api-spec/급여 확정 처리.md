# 급여 확정 처리

카테고리: 급여
설명: DRAFT 상태의 급여 내역을 CONFIRMED로 확정한다. 확정 이후에는 수정 및 재산출 불가.
Method: PUT
URL: /api/v1/salary/records/{id}/confirm
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "급여가 확정되었습니다." |
| data.id | 급여 내역 ID | Long |  | X | 1 |
| data.userId | 직원 ID | Long |  | X | 1 |
| data.userName | 직원 이름 | String |  | X | "박성종" |
| data.year | 년도 | Integer |  | X | 2026 |
| data.month | 월 | Integer |  | X | 4 |
| data.totalPay | 최종 지급액 (원) | Integer |  | X | 2650000 |
| data.status | 급여 상태 | String |  | X | "CONFIRMED" |
| data.updatedAt | 확정일시 | String |  | X | "2026-05-02T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "급여가 확정되었습니다.",
  "data": {
    "id": 1,
    "userId": 1,
    "userName": "박성종",
    "year": 2026,
    "month": 4,
    "totalPay": 2650000,
    "status": "CONFIRMED",
    "updatedAt": "2026-05-02T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 급여 확정 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 급여 내역 |
| 409 | 이미 확정된 급여 내역 |
