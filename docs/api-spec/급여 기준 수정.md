# 급여 기준 수정

카테고리: 급여
설명: 급여 기준을 수정한다. 이미 급여 산출에 사용된 기준은 수정 불가.
Method: PUT
URL: /api/v1/salary/settings/{id}
담당자: 박성종
사용자: ADMIN
우선순위: 6

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| baseSalary | 기본급 (원) | Integer |  | O | 2700000 |
| overtimeRate | 초과근무 배율 | Double |  | O | 1.5 |
| nightRate | 야간수당 배율 | Double |  | O | 2.0 |
| effectiveFrom | 적용 시작일 | String |  | O | "2026-04-01" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "baseSalary": 2700000,
  "overtimeRate": 1.5,
  "nightRate": 2.0,
  "effectiveFrom": "2026-04-01"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "급여 기준이 수정되었습니다." |
| data.id | 급여 기준 ID | Long |  | X | 1 |
| data.position | 직급명 | String |  | X | "사원" |
| data.baseSalary | 기본급 (원) | Integer |  | X | 2700000 |
| data.overtimeRate | 초과근무 배율 | Double |  | X | 1.5 |
| data.nightRate | 야간수당 배율 | Double |  | X | 2.0 |
| data.effectiveFrom | 적용 시작일 | String |  | X | "2026-04-01" |
| data.effectiveTo | 적용 종료일 | String |  | O | null |
| data.updatedAt | 수정일시 | String |  | X | "2026-04-01T10:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "급여 기준이 수정되었습니다.",
  "data": {
    "id": 1,
    "position": "사원",
    "baseSalary": 2700000,
    "overtimeRate": 1.5,
    "nightRate": 2.0,
    "effectiveFrom": "2026-04-01",
    "effectiveTo": null,
    "updatedAt": "2026-04-01T10:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 급여 기준 수정 성공 |
| 400 | 요청 값 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 급여 기준 |
| 409 | 이미 급여 산출에 사용된 기준으로 수정 불가 |
