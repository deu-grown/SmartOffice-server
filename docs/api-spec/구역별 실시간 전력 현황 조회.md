# 구역별 실시간 전력 현황 조회

카테고리: 전력관리
설명: 특정 구역의 가장 최근 1분 평균 전력 데이터를 조회한다. Redis 캐시에서 우선 조회하여 응답 속도를 최소화한다.
Method: GET
URL: /api/v1/power/zones/{zoneId}/current
담당자: 박성종
사용자: ADMIN
우선순위: 10

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data.zoneId | 구역 ID | Long |  | X | 1 |
| data.zoneName | 구역명 | String |  | X | "1층" |
| data.devices[].deviceId | 장치 ID | Long |  | X | 5 |
| data.devices[].deviceName | 장치명 | String |  | X | "1층 전력 센서" |
| data.devices[].avgWatt | 평균 전력 (W) | Double |  | X | 245.7 |
| data.devices[].measuredAt | 측정 일시 | String |  | X | "2026-04-01T14:23:00" |

**Example**

```json
{
  "code": "success",
  "message": "정상 조회되었습니다.",
  "data": {
    "zoneId": 1,

    "devices": [
      {
        "deviceId": 5,
        "deviceName": "1층 전력 센서",

      {

        "deviceName": "1층 서버실 전력 센서",
        "avgWatt": 820.3,

      }
    ]
  }
}
```

### Status

| status | response content |
| --- | --- |
| 200 | 실시간 전력 현황 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 |
