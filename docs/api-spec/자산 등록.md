# 자산 등록

카테고리: 자산관리
설명: 새로운 자산을 등록한다. allowedZoneId가 없으면 구역 제한 없이 사용 가능한 자산으로 등록된다.
Method: POST
URL: /api/v1/assets
담당자: 박성종
사용자: ADMIN
우선순위: 8

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| assetNumber | 자산 번호 | String |  | X | "ASSET-003" |
| name | 자산명 | String |  | X | "아이맥 27인치" |
| category | 자산 분류 | String |  | X | "ELECTRONICS" |
| rfidUid | RFID 태그 UID | String |  | O | "RF:G7:H8:I9" |
| allowedZoneId | 허용 구역 ID | Long |  | O | 2 |
| assignedUserId | 담당자 직원 ID | Long |  | O | 2 |
| description | 자산 설명 | String |  | O | "디자인팀 공용 iMac" |
| purchasedAt | 구입일 | String |  | O | "2026-01-10" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "assetNumber": "ASSET-003",
  "name": "아이맥 27인치",
  "category": "ELECTRONICS",
  "rfidUid": "RF:G7:H8:I9",
  "allowedZoneId": 2,
  "assignedUserId": 2,
  "description": "디자인팀 공용 iMac",
  "purchasedAt": "2026-01-10"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "자산이 등록되었습니다." |
| data.id | 생성된 자산 ID | Long |  | X | 3 |
| data.assetNumber | 자산 번호 | String |  | X | "ASSET-003" |
| data.name | 자산명 | String |  | X | "아이맥 27인치" |
| data.category | 자산 분류 | String |  | X | "ELECTRONICS" |
| data.rfidUid | RFID 태그 UID | String |  | O | "RF:G7:H8:I9" |
| data.allowedZoneId | 허용 구역 ID | Long |  | O | 2 |
| data.allowedZoneName | 허용 구역명 | String |  | O | "2층 개발팀" |
| data.assignedUserId | 담당자 직원 ID | Long |  | O | 2 |
| data.assignedUserName | 담당자 이름 | String |  | O | "김동우" |
| data.status | 자산 상태 | String |  | X | "ACTIVE" |
| data.purchasedAt | 구입일 | String |  | O | "2026-01-10" |
| data.createdAt | 등록일시 | String |  | X | "2026-04-01T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "자산이 등록되었습니다.",
  "data": {
    "id": 3,
    "assetNumber": "ASSET-003",
    "name": "아이맥 27인치",
    "category": "ELECTRONICS",
    "rfidUid": "RF:G7:H8:I9",
    "allowedZoneId": 2,
    "allowedZoneName": "2층 개발팀",
    "assignedUserId": 2,
    "assignedUserName": "김동우",
    "status": "ACTIVE",
    "purchasedAt": "2026-01-10",
    "createdAt": "2026-04-01T09:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 201 | 자산 등록 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 구역 ID 또는 직원 ID |
| 409 | 이미 등록된 자산 번호 |
| 409 | 이미 등록된 RFID UID |
