# 구역 등록

카테고리: 구역
설명: 새로운 구역을 등록한다. parentId가 없으면 최상위 구역으로 등록된다.
Method: POST
URL: /api/v1/zones
담당자: 박성종
사용자: ADMIN
우선순위: 4

### Request

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| name | 구역명 | String |  | X | "3층" |
| zoneType | 구역 유형 | String |  | X | "FLOOR" |
| parentId | 상위 구역 ID | Long |  | O | null |
| description | 구역 설명 | String |  | O | "3층 전체 구역" |

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

X

**Example (Request Body)**

```json
{
  "name": "3층",
  "zoneType": "FLOOR",
  "parentId": null,
  "description": "3층 전체 구역"
}
```

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "구역이 등록되었습니다." |
| data.id | 생성된 구역 ID | Long |  | X | 5 |
| data.name | 구역명 | String |  | X | "3층" |
| data.zoneType | 구역 유형 | String |  | X | "FLOOR" |
| data.parentId | 상위 구역 ID | Long |  | O | null |
| data.description | 구역 설명 | String |  | O | "3층 전체 구역" |
| data.createdAt | 생성일시 | String |  | X | "2026-04-01T09:00:00" |

**Example**

```json
{
  "code": "success",
  "message": "구역이 등록되었습니다.",
  "data": {
    "id": 5,
    "name": "3층",
    "zoneType": "FLOOR",
    "parentId": null,
    "description": "3층 전체 구역",
    "createdAt": "2026-04-01T09:00:00"
  }
}
```

### Status

| status | response content |
| --- | --- |
| 201 | 구역 등록 성공 |
| 400 | 요청 값 누락 또는 형식 오류 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
| 404 | 존재하지 않는 상위 구역 ID |
| 409 | 동일 상위 구역 내 중복 구역명 |
