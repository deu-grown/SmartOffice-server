# 현재 환경 현황 조회

카테고리: 대시보드
설명: 전체 구역의 현재 환경 센서 최신값을 조회한다. 구역별 온습도, CO2, 조도 데이터를 한번에 반환한다. Redis 캐시에서 우선 조회한다.
Method: GET
URL: /api/v1/dashboard/sensors/current
담당자: 박성종
사용자: ADMIN
우선순위: 7

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | 관리자 인증 토큰 (Bearer) | Bearer eyJhbGciOiJIUzI1Ni... |
| Content-Type | 데이터 전송 형식 | application/json |

**Query parameter**

X

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 응답 상태 | String | 필수 | No | "success" |
| message | 결과 메시지 | String | 필수 | No | "환경 현황 조회가 완료되었습니다." |
| data.sensorList | 각 구역별 센서 데이터 리스트 | Array | 필수 | No | [ { ... } ] |
| ...zoneName | 구역 이름 | String | 필수 | No | "A회의실" |
| ...temp | 현재 온도 (°C) | Double | 필수 | No | 24.5 |
| ...humi | 현재 습도 (%) | Double | 필수 | No | 45.0 |
| ...co2 | 이산화탄소 농도 (ppm) | Integer | 필수 | No | 420 |

**Example**

```json
{
  "status": "success",
  "message": "환경 현황 조회가 완료되었습니다.",
  "data": {
    "sensorList": [
      {
        "zoneId": 1,
        "zoneName": "A회의실",
        "temp": 24.5,
        "humi": 45,
        "co2": 400,
        "updatedAt": "2026-04-02T14:45:00"
      },
      {
        "zoneId": 2,
        "zoneName": "B회의실",
        "temp": 23.8,
        "humi": 42,
        "co2": 450,
        "updatedAt": "2026-04-02T14:44:10"
      }
    ]
  }
}
```

### Status

| status | response content | 설명 |
| --- | --- | --- |
| 200 | OK | 조회 성공 |
| 401 | Unauthorized | 관리자 인증 실패 |
| 503 | Service Unavailable | 센서 네트워크 또는 데이터베이스 연결 지연 |
