# 최근 출입 현황 조회

카테고리: 대시보드
설명: 최근 출입 이벤트 목록을 조회한다. 승인/거부 여부와 관계없이 최신순으로 반환한다. 기본 20건 반환.
Method: GET
URL: /api/v1/dashboard/access/recent
담당자: 박성종
사용자: ADMIN
우선순위: 7

### Request

X

**Header**

X

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| limit | 가져올 최근 기록의 개수 (기본값: 20) | Integer | Yes | 5 |
| type | 출입 타입 필터 (IN / OUT) | String | Yes | "IN" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| status | 응답 상태 | String | 필수 | No | "success" |
| message | 결과 메시지 | String | 필수 | No | "최근 출입 현황 조회가 완료되었습니다." |
| data.accessList | 출입 이력 데이터 배열 | Array | 필수 | No | [ { ... } ] |
| ...userName | 출입자 이름 | String | 필수 | No | "홍길동" |
| ...zoneName | 출입 구역 명칭 | String | 필수 | No | "제1회의실" |
| ...accessTime | 출입 발생 시각 | String | 필수 | No | "2026-04-02T14:30:00" |
| ...type | 출입 구분 (IN: 입실 / OUT: 퇴실) | String | 필수 | No | "IN" |

**Example**

```json
{
{
  "status": "success",
  "message": "최근 출입 현황 조회가 완료되었습니다.",
```

### Status

| status | response content | 설명 |
| --- | --- | --- |
| 200 | OK | 조회 성공 |
| 401 | Unauthorized | 관리자 인증 실패 |
| 404 | Not Found | 기록된 출입 이력이 없음 |
