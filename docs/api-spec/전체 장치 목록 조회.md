# 전체 장치 목록 조회

카테고리: 장치
설명: 전체 장치 목록을 조회한다. 구역, 장치 유형, 상태로 필터링 가능.
Method: GET
URL: /api/v1/devices
param: zoneId, deviceType, status
담당자: 우하
사용자: ADMIN
우선순위: 1

### Request

X

**Header**

| key | 설명 | 예시 |
| --- | --- | --- |
| Authorization | Bearer Access Token | "Bearer eyJhbGci..." |

**Query parameter**

| key | 설명 | 타입 | Nullable | 예시 |
| --- | --- | --- | --- | --- |
| zoneId | 구역 ID 필터 | Long | O | 1 |
| deviceType | 장치 유형 필터 | String | O | "NFC_READER" |
| status | 장치 상태 필터 | String | O | "ACTIVE" |

**Example (Request Body)**

X

### Response

| key | 설명 | value 타입 | 옵션 | Nullable | 예시 |
| --- | --- | --- | --- | --- | --- |
| code | 상태값 | String |  | X | "success" |
| message | 메시지 | String |  | X | "정상 조회되었습니다." |
| data[].id | 장치 ID | Long |  | X | 1 |
| data[].name | 장치명 | String |  | X | "1층 입구 NFC 리더기" |
| data[].deviceType | 장치 유형 | String |  | X | "NFC_READER" |
| data[].serialNumber | 시리얼 번호 | String |  | O | "SN-NFC-001" |
| data[].mqttTopic | MQTT 토픽 | String |  | O | "smartoffice/zones/1/devices/1/nfc" |
| data[].zoneId | 설치 구역 ID | Long |  | X | 1 |
| data[].zoneName | 설치 구역명 | String |  | X | "1층" |
| data[].status | 장치 상태 | String |  | X | "ACTIVE" |
| data[].createdAt | 등록일시 | String |  | X | "2026-03-02T09:00:00" |

### Status

| status | response content |
| --- | --- |
| 200 | 장치 목록 조회 성공 |
| 401 | Access Token 누락 또는 유효하지 않음 |
| 403 | 관리자 권한 없음 |
